Способи підключення стилів

  1. Зовнішні стилі (External CSS) - підключення стилів css, які зберігаються в окремому файлі
    <link rel="stylesheet" href="style.css">
  2. Внутрішній CSS (Inline CSS) - стилі вписуються прямо в атрибут style HTML-елемента.
    <p style="color: blue;">Текст</p>
  3. Вбудовані стилі (Internal CSS) - стилі розміщуються у тегу <style> в <head> секції документа. 
    <style>p { color: blue; }</style>
  4. Імпорт CSS - використання @import у CSS-файлі для підключення інших CSS-файлів.
    @import url("style.css");
