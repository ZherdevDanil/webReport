Вставка(тег script)

Тег <script> в HTML використовується для додавання та виконання JavaScript-коду на веб-сторінці. Він дозволяє розробникам вставляти скрипти безпосередньо в HTML-документ або посилатися на зовнішні JavaScript-файли.

    <script>
        // Приклад вбудованого JavaScript-коду
        console.log('Hello, world');
    </script>
