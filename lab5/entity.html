Підстановка в JavaScript

Підстановка в JavaScript - це процес заміни одного виразу або значення на інше. Це важливий механізм для створення динамічних шаблонів або вставки змінних значень у рядки.

Наприклад, ви можете використовувати підстановку для створення рядка, який містить значення змінних. Це зручно для форматування тексту або відображення динамічної інформації.

Приклад підстановки в JavaScript


                

    let city = 'Київ';
    let temperature = 25;

    // Використання підстановки для створення рядка зі значеннями змінних
    let weatherReport = `Сьогодні в ${city} температура становить ${temperature} градусів.`;

    console.log(weatherReport); // виведе "Сьогодні в Київ температура становить 25 градусів."
    

                
                    
У цьому прикладі підстановка використовується для вставки значень змінних city та temperature у рядок weatherReport. В результаті ми отримуємо рядок, який містить конкретні значення змінних.

                

            
Розширене використання підстановки

            
Підстановка в JavaScript також дозволяє виконувати вирази всередині шаблонних рядків. Це може бути
                корисним для виконання обчислень або виклику функцій безпосередньо в рядку.

            

    let product = 'яблука';
    let price = 10;
    let quantity = 5;

    // Використання підстановки для виконання обчислень всередині шаблонного рядка
    let total = `Загальна вартість ${quantity} ${product} становить ${price * quantity} гривень.`;

    console.log(total); // виведе "Загальна вартість 5 яблук становить 50 гривень."
    


            
Підстановка з функціями

            
Ви можете використовувати функції всередині шаблонних рядків для динамічного формування тексту.

            

    function greet(name) {
        return `Привіт, ${name}!`;
    }

    let userName = 'Олена';

    // Використання функції всередині шаблонного рядка
    let greeting = `${greet(userName)}`;

    console.log(greeting); // виведе "Привіт, Олена!"
