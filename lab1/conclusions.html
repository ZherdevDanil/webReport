Висновки

Під час виконання лабораторної роботи були виконані такі дії

  • Було обрано предметну галузь майбутнього web застосунку
  • Був написаний HTML-код базової сторінки звітного документу, який слугує основою для подальшої розробки та стилізації.
  • За допомогою використання CSS розроблені html сторінки було стилізовано, що надало їм привабливого та професійного вигляду.
  • Були вивчені та використанні на практиці такі елементи веб сторінки, як посилання, таблиці, форми, зображення,списки та заголовки різних рівнів
  • Використовуючи GitHub Pages були задеплоєні розроблені web застосунки

Підсумовуючи можна сказати, що завдяки виконання цієї лабораторної роботи були вивчені елементи html сторінок, а також покращені навички верстки, окрім використання html були застосовані каскадні таблиці стилів для стилізації написах сторінок.
