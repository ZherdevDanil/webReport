Проект складається з наступних компонентів:
  • HTML-файл (index.html):
    • Визначає структуру веб-сторінки.
    • Містить метатеги для кодування, перегляду та заголовок сторінки.
    • Підключає зовнішні ресурси, такі як шрифти та стилі.
    • Містить заголовок сторінки, зображення символу КПІ та інформацію про студента та групу.
    • Включає списки лабораторних робіт та контент для лабораторних робіт.

  • CSS-файл (styles.css): - знаходиться в папці css
    • Містить стилі для оформлення елементів веб-сторінки, таких як розмір шрифту, кольори, відступи тощо.

  • Сховище для зображень: папка images
    • Містить всі медіафайли необхідні для написання звітів
